SQL 高级语法教程(MySQL用法) 一

1.SQL TOP 子句：

TOP 子句用于规定要返回的记录的数目。

SELECT TOP 数量|百分比 列名称
FROM 表名称

MySQL 语法:

SELECT 列名称
FROM 表名称
LIMIT 起始行, 最多行数 (行数 0 开始 算)

2.SQL LIKE 操作符：

LIKE 操作符用于在 WHERE 子句中搜索列中的指定模式。

SELECT 列名称
FROM 表名称
WHERE 列名称 LIKE 条件

“%” 可用于定义通配符（模式中缺少的字母）。

LIKE 'N%'           (以 "N" 开始)

LIKE '%g'           (以 "g" 结尾)

LIKE '%lon%'        (包含 "lon")

NOT LIKE '%lon%'    (不包含 "lon")

3.SQL 通配符：

在搜索数据库中的数据时，SQL 通配符可以替代一个或多个字符。
PS：SQL 通配符必须与 LIKE 运算符一起使用。

通配符 描述
% 替代一个或多个字符
_ 仅替代一个字符
[charlist] 字符列中的任何单一字符
[^charlist]或者[!charlist] 不在字符 
[charlist] ：LIKE '[ALN]%'      (以 "A" 或 "L" 或 "N" 开头)

[!charlist] ：LIKE '[!ALN]%'    (不以 "A" 或 "L" 或 "N" 开头)

4.SQL IN 操作符：

IN 操作符允许我们在 WHERE 子句中规定多个值。

SELECT 列名称
FROM 表名称
WHERE 列名称 IN (条件1,条件2,...)

5.SQL BETWEEN 操作符：

操作符 BETWEEN … AND 会选取介于两个值之间的数据范围。这些值可以是数值、文本或者日期。

SELECT 列名称
FROM 表名称
WHERE 列名称
BETWEEN 条件1（包括） AND 条件2（不包括）
BETWEEN 条件1（包括） AND 条件2（不包括）

NOT BETWEEN 条件1（不包括） AND 条件2（包括）

6.SQL Alias（别名）：

可以为列名称和表名称指定别名（Alias）。
表的别名:

SELECT 列名称
FROM 表名称 AS 别名

列的别名：

SELECT 列名称 AS 别名
FROM 表名称

7.SQL JOIN：

用于根据两个或多个表中的列之间的关系，从这些表中查询数据。
不同的 SQL JOIN：

  • JOIN: 如果表中有至少一个匹配，则返回行
  • LEFT JOIN: 即使右表中没有匹配，也从左表返回所有的行
  • RIGHT JOIN: 即使左表中没有匹配，也从右表返回所有的行
  • FULL JOIN: 只要其中一个表中存在匹配，就返回行

1.INNER JOIN 关键字：

在表中存在至少一个匹配时，INNER JOIN 关键字返回行。

SELECT 列名称
FROM 表1
INNER JOIN 表2
ON 表1.列=表2.列

2.LEFT JOIN 关键字：

LEFT JOIN 关键字会从左表 (表1) 那里返回所有的行，即使在右表 (表2) 中没有匹配的行。

SELECT 列名称
FROM 表1
LEFT JOIN 表2
ON 表1.列=表2.列

3.RIGHT JOIN 关键字：

RIGHT JOIN 关键字会右表 (table_name2) 那里返回所有的行，即使在左表 (table_name1) 中没有匹配的行。

SELECT 列名称
FROM 表1
RIGHT JOIN 表2
ON 表1.列=表2.列

4.FULL JOIN 关键字：

只要其中某个表存在匹配，FULL JOIN 关键字就会返回行。

SELECT 列名称
FROM 表1
FULL JOIN 表2
ON 表1.列=表2.列

8.SQL UNION 和 UNION ALL 操作符：

UNION 操作符用于合并两个或多个 SELECT 语句的结果集。
UNION 语法：

SELECT 列名称 FROM 表1
UNION
SELECT 列名称 FROM 表2

UNION ALL 命令和 UNION 命令几乎是等效的，不过 UNION ALL 命令会列出所有的值。
UNION ALL 语法：

SELECT 列名称 FROM 表1
UNION ALL
SELECT 列名称 FROM 表2

9.SQL SELECT INTO 语句：

可用于创建表的备份复件。

SELECT 列名称
INTO 新表 [ IN 数据库 ]
FROM 老表

10.SQL CREATE DATABASE 语句：

SQL CREATE DATABASE 语句

CREATE DATABASE 表名称

11.CREATE TABLE 语句：

语句用于创建数据库中的表。

CREATE TABLE 表名称
(
列名称1 数据类型,
列名称2 数据类型,
列名称3 数据类型,
....
)
